# @sequential/desktop-ui-components

Reusable UI components library for Sequential Desktop applications with shared styling and functionality.

## Installation

```bash
npm install @sequential/desktop-ui-components
```

## Usage

```javascript
import {{ main }} from '@sequential/desktop-ui-components';

const result = main(/* args */);
```

## API

### Main Export

Main function/class provided by this package.

## Examples

```javascript
// Basic usage
const value = main(input);

// With options
const result = main(input, {{ /* options */ }});
```
